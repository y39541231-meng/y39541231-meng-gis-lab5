# GIS Lab 2025/26 – Mapping Air Quality in Italy

This repository contains the website and WebGIS code for **Group 6** of the GIS Lab project.

The project analyzes air pollution change, land-cover transition, and population exposure in Italy between 2021 and 2023.

## Pages

* `index.html` – project overview, study area, data sources, and team contribution
* `methods.html` – processing workflow and main GIS methods
* `results.html` – tables, charts, maps, and result interpretation
* `webgis.html` – WebGIS page

## Main data themes

* CAMS air quality data: NO2, PM2.5, PM10
* ESRI 10 m Annual Land Cover: 2021 and 2023
* WorldPop population count: 2023
* Italy case-study boundary and administrative zones
